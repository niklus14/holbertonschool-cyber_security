Introduction
Web applications often execute instructions blindly, hiding risks in plain sight. This project moves beyond surface-level usage, teaching students to think like investigators. By tracing requests and analyzing unexpected behaviors, you will learn to identify hidden vulnerabilities, transforming simple curiosity into professional security analysis.
